Ready Rent Cars

Acerca de este proyecto

Proyecto desarrollado con el fin de digitalizar una rentadora de autos, la cual solicitaba un medio por el cual redireccionar y comunicar a sus usuarios a sus canales principales de venta.

Sitio Web informativo que redirecciona a los usuarios a canales principales de comunicación.

Tecnologías implementadas

  • HTML5 (Estructuración)
  • CSS3 (Estilos)
  • JavaScript (Lenguaje Programación)
  • VsCode (Editor de código)
  • Git (Gestión de versiones)
