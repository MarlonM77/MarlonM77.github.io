Gestionation S.A.S

Acerca de este proyecto

Proyecto desarrollado mediante una arquitectura MVC para la gestión de usuarios en una empresa.

Programa de escritorio capaz de crear-consultar-eliminar-actualizar todo tipo de usuarios para el gestionamiento de sus datos en una empresa.

Tecnologías implementadas

  • Java (Lenguaje Programación)
  • MySQL (BD)
  • NetBeans (Editor de código)
  • Git (Gestión de versiones)
