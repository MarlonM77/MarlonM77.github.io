Viaja Por Colombia

Acerca de este proyecto

Proyecto desarrollado mediante arquitectura de 3 capas, donde se diseñó y desarrolló desde su base de datos, su capa lógica hasta su capa de presentación.

Sitio web de turismo capaz de crear y autenticar usuarios que puedan ver y reservar sus planes de turismo.

Tecnologías implementadas

  • VueJs (Frontend)
  • Nodejs (Levantamiento Fe)
  • Django Rest (Backend)
  • PostgreSQL (BD)
  • Postman (Testing)
  • Heroku (Despliegue)
  • Git (Gestión de versiones)
